    What is Rock API?

    Rock API is an API that gives you ROCKS. However, this isn't a regular "rock finding" API... We have rocks like, glow rock, falling rock, and MANY more. This api was made for entertainment purposes. Use this API in your Discord bot for commands like pls rock or !randomrock or even in your website for... something.

    API Docs

    Covering different endpoints

    Get Random Rock

    URL: https://rockapi.apiworks.tech/rock/random

    Returns a random rock in JSON format with the following keys: name, description, and image_url, all of which are strings.

    TypeReturnsOutput ExampleSource
    GET200 or 404{"name":"rock",
    "description":"description",
    "image_url":"img url"}    		Link

    Get Random Rock with Tag

    URL: https://rockapi.apiworks.tech/rock/random?tags=tag here

    Add tags separated by a comma. Ex: rock,party,pun. If a tag is not found, it will return a 404.

    TypeReturnsInput ExampleSource
    GET200 or 404/rock/random?tags=rock,party,punLink

    Search Rock

    URL: https://rockapi.apiworks.tech/rock/rock here

    If a rock is not found, it will return a 404.

    TypeReturnsInput ExampleSource
    GET200 or 404/rock/music rockLink
